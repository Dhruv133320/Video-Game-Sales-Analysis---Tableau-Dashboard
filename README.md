# 🎮 Video Game Sales Analysis using Tableau

## Overview

This project presents an interactive Tableau dashboard built to analyze the global video game market. The dashboard provides insights into game sales across different regions, platform performance, genre popularity, publisher and developer statistics, critic and user ratings, and historical sales trends.

The objective of this project is to transform raw sales data into meaningful business insights through interactive visualizations.

---

## Dataset

The project uses a Video Games Sales dataset containing information such as:

* Game Name
* Platform
* Genre
* Publisher
* Developer
* Year of Release
* Critic Score
* User Score
* Regional Sales
* Global Sales
* Age Rating

---

## Dashboard Features

### Key Performance Indicators (KPIs)

* Total Global Sales
* Average Critic Score
* Average User Score
* North America Sales
* Europe Sales
* Japan Sales
* Other Region Sales

### Visualizations

* Top 5 Best Selling Games
* Top 10 Games by User Count
* Platform vs Global Sales
* Genre-wise Sales Distribution
* Global Sales Trend Over Time
* Average Critic Score vs User Score by Genre
* Number of Games by Publisher
* Number of Games by Developer
* Number of Games by Age Rating
* Age Rating vs Global Sales

---

## Business Insights

* Identify the highest-selling video games globally.
* Compare sales performance across different gaming platforms.
* Understand genre popularity based on total sales.
* Analyze regional sales distribution.
* Compare critic ratings with user ratings.
* Discover the most active publishers and developers.
* Observe historical sales trends over the years.
* Evaluate sales across different age rating categories.
